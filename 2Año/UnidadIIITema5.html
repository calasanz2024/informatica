Salidas Analógicas

A diferencia de las salidas digitales que solo pueden estar en "alto" (5V) o "bajo" (0V), las salidas analógicas permiten enviar un rango de valores entre 0 y 255 (en el caso de Arduino) para simular voltajes intermedios. Aunque las salidas de Arduino no son realmente analógicas (son señales PWM, o Modulación por Ancho de Pulso), se les llama así porque pueden imitar un rango de voltajes.

Las salidas analógicas son útiles cuando quieres controlar dispositivos que no solo necesitan estar encendidos o apagados, sino que requieren un control más gradual, como la intensidad de un LED, la velocidad de un motor, o la posición de un servo.

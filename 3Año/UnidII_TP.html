Trabajo Práctico

Diseñar y construir un circuito electrónico que combine el uso de un pulsador, tres LED (Rojo-Amarillo-Verde), un buzzer, un sensor de temperatura y un LDR (sensor de luz). El sistema debe poder monitorear la temperatura y la luz ambiental, y alertar cuando ciertos umbrales sean alcanzados. Además, el sistema debe permitir la activación manual de un estado de alerta mediante el pulsador.

Ten en cuenta lo siguiente:

  1. Leer los valores del sensor de temperatura y el LDR.
  2. Establecer umbrales para la temperatura y la luz:
    • Si la temperatura supera los 30°C, enciende el LED rojo, activa el buzzer y muestra en pantalla: ¡Calor!
    • Si la luz ambiental es menor que 200, enciende el LED amarillo y muestra en pantalla: ¡Oscuro!
  3. Si se presiona el pulsador, activa un estado de alerta que encienda todos los LEDs, el buzzer y muestre en pantalla: ¡Alerta!
  4. Utiliza el LED verde para indicar que el sistema está funcionando correctamente cuando no hay alertas y muestre en pantalla: ¡Optimo!
